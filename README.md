#### GATE API EMULATOR

Для того что бы данные в sqlite сохранялись при запуске через докер необходимо в корне проекта создать файл `gatemulator.db`
В противном случае данные будут сохраняться в памяти и при остановки приложения данные потеряются.
На текущий момент сохраняются subscribers и subscriptions.
